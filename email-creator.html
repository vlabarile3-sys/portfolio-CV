Quick Guide

Core Concept

Viriathus Email Creator is a batch automation engine. Select multiple agents and automatically generate personalized email drafts with optional Drive file duplication — all in a single execution.

The 3-Step Process

  • 1. Select Workflow: Choose the target group.
  • 2. Select Agents: Use the multi-select dropdown. Search by name, or use 'Select All'.
  • 3. Proceed & Confirm: Pick a template from the Wizard. Review the live preview, then click Generate.

Database Management

Open "Advanced Database Management" to add agents or create new workflow groups.

Template Manager

Click the spinning gear icon to create and manage reusable email templates with dynamic variables.

Pro Tip: The "Drive File ID" is optional. If provided, the system duplicates it for each agent. If blank, only the email is generated.
